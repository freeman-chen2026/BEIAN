# 每日通航运行情况跟踪表生成器（纯文本版）

**功能**：上传航段数据 Excel，生成制表符分隔的表格文本，复制粘贴到 Excel 即可完成备案。

**特点**：
- 无需模板，不依赖样式复制，稳定可靠。
- 日期格式：`2026/7/8`
- 飞行开始时间：优先“实际出发”，若无则用“计划出发”
- 机型代码自动映射（可侧边栏自定义）
- 运行种类自动判断
- 按原始顺序排列，不重排。

**使用方法**：
1. 上传 `航段数据导出 (xxx).xlsx`
2. 在“复制以下文本”框中全选复制
3. 打开 Excel，点击 A1 单元格，粘贴（若未自动分列，使用“数据”->“分列”，分隔符选“制表符”）
4. 调整格式后即可用于备案。

**也可下载 CSV 备用**。
