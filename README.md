# 每日通航运行情况跟踪表生成器

将“航段数据导出”格式的 Excel 转换为监管局备案格式。

## 功能亮点
- 自动识别表头（跳过空行）
- 航线显示为 **出发城市 - 到达城市**（例如“香港-广州白云”）
- 根据注册号自动填入 **ICAO 机型代码**（内置您指定的映射，侧边栏可自定义补充）
- 导出格式与监管局模板完全一致

## 使用
1. 上传 Excel
2. 如需新增注册号-机型映射，在侧边栏“自定义映射”文本框中按 `注册号 ICAO` 格式添加（每行一个）
3. 预览无误后下载 Excel 或 CSV
