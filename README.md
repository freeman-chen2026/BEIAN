# 每日通航运行情况跟踪表生成器

**使用步骤**：
1. **首次使用**：上传您的带格式模板（例如 `每日通航运行情况跟踪表（天成商务航空有限公司）20260707.xlsx`），程序会缓存模板样式。
2. 以后每次只需上传 `航段数据导出 (xxx).xlsx` 数据文件。
3. 点击下载生成的 Excel，所有样式（底色、字体、边框、列宽）与模板完全一致，数据自动填充。

**特点**：
- 模板只需上传一次，后续自动复用。
- 自动识别表头，读取航段数据。
- 自动映射机型代码。
- 根据“航段状态”自动判断是否落地并填入实际时间。
- 日期格式严格按照模板（`YYYY-MM-DD 00:00:00`），“选择允许的运行种类”自动填入 `3.公务航空运行`。
- 所有单元格样式复制自模板表头行，确保格式统一。
